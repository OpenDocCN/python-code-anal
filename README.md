## 下载

### Docker

```
docker pull apachecn0/python-code-anal
docker run -tid -p <port>:80 apachecn0/python-code-anal
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install python-code-anal
python-code-anal <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g python-code-anal
python-code-anal <port>
# 访问 http://localhost:{port} 查看文档
```
